### Descripción del proyecto:
Para la construcción de este sitio web utilizamos HTML5 siguiendo una estructura semántica clara (`<header>`, `<nav>`, `<main>`, `<footer>`). El enfoque principal fue maquetar la plataforma de "Gestión de Proyectos Educativos" vinculando cuatro páginas estáticas sin el uso de CSS, cumpliendo con los requisitos de la primera etapa del Trabajo Práctico.

En esta segunda etapa del Trabajo Práctico, implementamos una hoja de estilos externa (styles.css) basada en variables, lo que nos permitió aplicar una paleta de colores y tipografía coherente en todas las páginas.Donde vinculamos todos los archivos HTML al mismo CSS y la navegación sean idénticos en todo el sitio.
