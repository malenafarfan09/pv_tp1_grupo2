Plataforma Educativa Web

Fecha de inicio: 2026

Autores: Grupo de Programación Visual

Descripción

Este proyecto tiene como objetivo el desarrollo de una plataforma web educativa que permita gestionar proyectos académicos de manera eficiente. La plataforma facilitará la organización de tareas, seguimiento de avances y comunicación entre los integrantes.

Se implementarán tecnologías web como HTML y CSS para la estructura y diseño del sitio. Además, se busca que la interfaz sea clara, accesible y fácil de usar para estudiantes y docentes.
